# Legal Rights Website

An interactive and beginner-friendly website aimed at empowering Indian citizens with knowledge about their legal rights and helping them report bribery or harassment safely.

## Features

1. **Homepage**
   - Quick access buttons: Know My Rights, Bribe/Harassment Map, Emergency Call.
   - Pastel, soft design with cute icons.
   - Fully mobile responsive.

2. **Know My Rights Page**
   - Lists common legal situations (Traffic Stop, FIR, Detention, Female Arrest, Phone Check).
   - Multilingual support: English, Hindi, Tamil, Telugu, Marathi.
   - Interactive layout with a soft, professional design.

3. **Bribe/Harassment Map**
   - Users can click on the map to report incidents (Bribe or Harassment).
   - Reports stored locally and displayed as map markers with popups.
   - Uses Leaflet.js with OpenStreetMap (no API keys required).

4. **Emergency Call**
   - Quick button to call 100 on mobile devices.

## Technologies Used

- HTML, CSS, JavaScript
- Leaflet.js for interactive maps
- LocalStorage for storing reports
- Font Awesome for icons
- Responsive design for mobile and desktop
